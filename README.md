# StartupIQ - AI-Powered Startup Feasibility Analyzer

**StartupIQ** is an intelligent "Virtual Co-Founder" platform designed to help student entrepreneurs validate their business ideas instantly. It utilizes a **Hybrid AI Architecture** that combines **Generative AI (Google Gemini)** for qualitative SWOT analysis and **Statistical Machine Learning (Linear Regression)** for quantitative financial valuation.

* **Hybrid Intelligence:** Combines LLM reasoning (Gemini) with a mathematical regression model to prevent "hallucinated" financial metrics.
* **Smart Fallback Mechanism:** A resilient backend system that automatically switches to a local logic engine if the external AI API fails, ensuring **100% uptime** during demos.
* **Parallel Processing:** Implements multi-threading in Flask to execute AI analysis and financial calculations simultaneously, reducing wait times by ~40%.
* **Interactive Dashboard:** Visualizes Feasibility Scores, SWOT analysis, and Financial Projections.
* **Instant Reports:** Client-side PDF generation allowing users to download professional investor-ready reports.
